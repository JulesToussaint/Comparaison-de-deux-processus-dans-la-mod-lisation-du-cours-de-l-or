# Comparaison-de-deux-processus-dans-la-mod-lisation-du-cours-de-l-or
L'or est coincé entre deux classes : commodity et actif d'investissement. Pour palier à cette ambiguïté, ce notebook inspecte le comportement du cours de l'or et le compare à deux processus classiques dans la modélisation d'une commodity et d'un actif d'investissement.
